# システム概要ページ 実装設計書

## 概要

**目的:** 投資家・上司へのデモ用に、VATCH が SES 業務のエンドツーエンドをどう自動化するかを一目で伝えるページを追加する。

**配置:** サイドバーの「システム概要」メニューから遷移する専用ページ（`/overview`）。

## デザイン仕様

- **テーマ:** 既存のダーク系（`#0d1117` ベース）に準拠
- **サイドバー:** 既存コンポーネントを再利用。「システム概要」をアクティブ状態にする
- **レイアウト:** 既存ダッシュボードと同じサイドバー + メインエリア構造
- **フロー部分:** 横スクロール対応（`overflow-x: auto`）

---

## ファイル構成

| ファイル | 役割 |

| `src/app/(dashboard)/overview/page.tsx` | ページコンポーネント（Server Component） |

## 実装上の注意

- 静的コンテンツのみ（API 呼び出しなし）
- `'use client'` 不要（Server Component）
- 既存のレイアウトコンポーネント（`(dashboard)/layout.tsx`）を継承
